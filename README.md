# Finite_Element_Method_Project01
FINITE ELEMENT ANALYSIS(FEM)


# Cantilever Beam Deflection using Finite Element Method(FEM) and Comparison with Analytical Solution
<b> 1 Aim</b>
<br>
To determine the deflection of a cantilever beam subjected to (a) a point load at the free end
and (b) a uniformly distributed load (UDL), using the Finite Element Method with Hermite beam
elements, and to compare the results with analytical solutions from Strength of Materials.
<br>
2 Objectives
1. To model a cantilever beam using Euler–Bernoulli beam theory.
2. To assemble the element stiffness matrix for Hermite beam elements.
3. To apply boundary conditions and external loads, and to impose boundary conditions
4. To obtain nodal displacements and reconstruct the beam deflected shape.
5. To compare FEM results (deflections and convergence) with analytical solutions.
6. To study mesh convergence.
